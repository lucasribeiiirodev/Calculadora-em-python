# 🧮 Calculadora em Python

Uma calculadora desenvolvida com Python e Tkinter, com interface gráfica moderna e lógica própria para cálculos matemáticos.

## 🚀 Funcionalidades

- Operações matemáticas:
  - Soma (+)
  - Subtração (-)
  - Multiplicação (×)
  - Divisão (÷)

- Interface gráfica com Tkinter
- Separação da lógica em módulos
- Sistema de cores personalizado
- Tratamento de erros
- Prioridade de operadores (* e /)
- Código organizado para estudos e evolução do projeto

---

## 📂 Estrutura do Projeto

```bash
Calculadora-em-python/
│
├── main.py        # Inicialização do projeto

├── logica.py      # Lógica dos cálculos

└── README.md
```

## 🛠 Tecnologias Utilizadas

- Python 3

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone <PRIVATE_URL>
```

2. Acesse a pasta:

```bash
cd Calculadora-em-python
```

3. Execute o projeto:

```bash
python main.py
```
